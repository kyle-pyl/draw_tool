# 用户手册

本手册面向人类用户，介绍绘图工具的基本使用方法，包括安装、打开项目、绘制图形、编辑、数据图表和导出等操作。

## 1. 安装和启动

### 1.1 安装

本工具为纯 Web 应用，无后端依赖。

```bash
# 克隆项目后安装依赖
npm install

# 开发模式启动
npm run dev

# 构建生产版本
npm run build        # Full 包（含 Excel 支持）
npm run build:lite   # Lite 包（轻量版本）
npm run build:all    # 同时构建两个版本
```

### 1.2 启动

- **开发模式**：`npm run dev` 启动 Vite 开发服务器，浏览器访问 `http://localhost:5173`
- **生产模式**：打开 `dist/full/` 或 `dist/lite/` 目录中的 `index.html`
- **PWA 离线**：首次访问后，页面可离线使用。浏览器地址栏出现安装图标时可安装为桌面应用

### 1.3 双版本说明

| 特性 | Lite 包 | Full 包 |
|------|---------|---------|
| SVG 画布、图形绘制、文本、图片 | 支持 | 支持 |
| 图层管理、碰撞检测、冲突高亮 | 支持 | 支持 |
| 连接线、锚点、箭头、标签 | 支持 | 支持 |
| 模板系统、流程图、架构图、RTL | 支持 | 支持 |
| CSV 数据解析和图表生成 | 支持 | 支持 |
| Excel (.xlsx/.xls) 导入 | 不支持 | 支持 |
| 自动布局（排版） | 基础算法 | 完整算法 |
| 导出 SVG/PNG/JPG | 支持 | 支持 |

## 3. 打开和保存项目

### 3.1 打开项目

**方式一：打开 JSON 文件**
1. 点击菜单栏 文件 > 打开文件
2. 选择本地 `scene.json` 文件
3. 工具自动校验文件，通过后渲染到画布

**方式二：打开项目目录（Chrome/Edge）**
1. 点击菜单栏 文件 > 打开项目目录
2. 选择包含 `scene.json` 的项目文件夹
3. 工具自动读取场景、数据文件和资源

**方式三：导入 ZIP**
1. 点击菜单栏 文件 > 导入 ZIP
2. 选择项目 ZIP 包（内部结构为 scene.json + data + assets）
3. ZIP 为浏览器通用导入方式，不依赖 File System Access API

**方式四：从示例项目打开**
1. 点击菜单栏 文件 > 打开示例 > 选择示例项目
2. 预置示例：基础图形、流程图、架构图、RTL、统计图、网络拓扑

### 3.2 保存项目

- **Ctrl+S** 或 文件 > 保存
- 若有目录句柄，直接写入 scene.json
- 若无目录句柄，自动触发 ZIP 下载

### 3.3 关闭项目

- 文件 > 关闭项目（如在修改未保存时提示）

## 4. 画布操作

### 4.1 视口操作

| 操作 | 方法 |
|------|------|
| 缩放 | 鼠标滚轮（以鼠标位置为中心缩放） |
| 平移 | 空格键 + 鼠标拖拽，或鼠标中键拖拽 |
| 适配画布 | 菜单栏 视图 > 适配画布 |
| 适配选区 | 选中元素后 视图 > 适配选区 |
| 缩放到 100% | 视图 > 实际大小 |
| 重置视图 | 视图 > 重置视图 |

### 4.2 网格和标尺

| 功能 | 操作 |
|------|------|
| 显示/隐藏网格 | 视图 > 显示网格 |
| 显示/隐藏标尺 | 视图 > 显示标尺 |
| 吸附到网格 | 默认启用，移动元素时自动吸附 |
| 临时禁用吸附 | 按住 **Alt** 键移动元素 |

### 4.3 选择元素

| 操作 | 方法 |
|------|------|
| 单选 | 点击元素 |
| 多选 | **Shift** + 点击多个元素 |
| 框选 | 在空白区域按住鼠标拖拽绘制选择框 |
| 追加框选 | **Shift** + 框选 |
| 全选 | **Ctrl+A** |
| 取消选择 | 点击空白区域或按 **Esc** |
| 按图层全选 | 图层面板中点击图层行 |

### 4.4 选中元素的外观

选中元素显示：
- 蓝色包围盒（8 个控制柄：四角 + 四边中点）
- 旋转控制柄（上方延伸点）

## 5. 绘制图形

### 5.1 形状工具

左侧工具栏选择形状类型，在画布上拖拽绘制：

| 形状 | 绘制方式 | 说明 |
|------|----------|------|
| 矩形 | 拖拽对角点 | 支持圆角（属性面板设置） |
| 圆 | 拖拽外接矩形 | `Shift` 约束为正圆 |
| 椭圆 | 拖拽外接矩形 | `Shift` 约束为正圆 |
| 多边形 | 连续点击顶点，双击闭合 | 至少 3 个顶点 |
| 路径 | 连续点击添加线段，双击结束 | 支持自由路径 |
| 线条 | 拖拽起止点 | - |

### 5.2 文本工具

1. 选择文本工具，点击画布创建文本框
2. 双击已有文本进入编辑模式
3. 在属性面板或编辑框中设置：字体、字号、加粗、斜体、颜色、对齐
4. 支持边框和背景底纹

### 5.3 图片导入

1. 工具 > 导入图片，或直接拖放图片文件到画布
2. 支持的格式：**PNG、JPG、SVG**
3. SVG 导入时自动进行安全清洗（移除脚本等危险元素）
4. 导入后可移动、缩放

### 5.4 连接线工具

1. 选择连接线工具
2. 鼠标悬停元素时显示可用锚点（上下左右四方向 + 四角）
3. 从源元素的锚点拖拽到目标元素的锚点创建连接线
4. 连接线路由类型：直线、折线、正交线
5. 属性面板设置：箭头样式、线条颜色、虚线、标签

### 5.5 模板插入

1. 打开模板面板（右侧浮动面板）
2. 按分类浏览：基础几何、流程图、架构图、RTL、论文图、思维导图、拓扑图
3. 搜索框按名称模糊搜索
4. 点击模板缩略图插入到画布中心

## 7. 图层管理

### 7.1 图层概念

- 图层按 **order** 决定渲染顺序，order 值大的图层在上方
- 普通元素在同层内**不允许重叠**（按包围盒碰撞检测）
- 连接线是特殊元素，**不受同层不可重叠约束**，可跨层连接
- 锁定元素不可编辑，但仍参与冲突检测
- 隐藏元素默认参与占位（避免重新显示时破坏约束）

### 7.2 图层面板操作

右侧图层面板显示所有图层：

| 操作 | 方法 |
|------|------|
| 可见性切换 | 点击眼睛图标 |
| 锁定切换 | 点击锁图标 |
| 选中全层元素 | 点击图层行 |
| 拖拽排序 | 拖拽图层行（通过 MoveLayers 命令执行） |
| 批量操作 | 右键图层行或点击更多按钮可进行：全选、统一改色、统一透明度、显示/隐藏全部、复制到其他层、移动到其他层、删除全部 |

### 7.3 冲突处理

- 当操作导致同层元素重叠时，操作被阻止
- 冲突元素显示**红色虚线包围盒**
- 图层面板中冲突图层标红
- 冲突面板显示具体错误信息：冲突层、冲突元素、建议修复方案

## 8. 对齐和分布

### 8.1 对齐

选中 2 个及以上元素后，在对齐工具栏中选择对齐方式：

| 对齐类型 | 效果 |
|----------|------|
| 左对齐 | 所有元素左边缘与最左元素对齐 |
| 右对齐 | 所有元素右边缘与最右元素对齐 |
| 上对齐 | 所有元素上边缘与最上元素对齐 |
| 下对齐 | 所有元素下边缘与最下元素对齐 |
| 水平居中 | 所有元素水平中心点对齐 |
| 垂直居中 | 所有元素垂直中心点对齐 |
| 中心对齐 | 所有元素中心点重合 |

对齐操作会检查图层冲突，存在冲突时阻止并高亮。

### 8.2 分布

| 分布类型 | 效果 |
|----------|------|
| 水平均匀 | 选中元素在水平方向等间距排列 |
| 垂直均匀 | 选中元素在垂直方向等间距排列 |
| 环形排列 | 选中元素绕指定圆心等距等弧度排列 |

### 8.3 分组

- 选中元素后按 **Ctrl+G** 创建组
- 分组可跨图层
- 按 **Ctrl+Shift+G** 解散组
- 点击组成员时提供"选中整组"操作

## 9. 数据图表

### 9.1 导入数据

1. 文件 > 导入数据，选择 CSV 文件
2. Excel 文件（.xlsx/.xls）仅在 Full 包中支持
3. 导入后数据面板显示数据源信息

### 9.2 数据面板

| 信息 | 说明 |
|------|------|
| 数据源列表 | 显示当前项目加载的数据源 |
| 列预览 | 列名、推断类型（number/string/date/boolean）、缺失率 |
| 样例行 | 前几行数据预览 |

### 9.3 创建图表

1. 在数据面板中选择数据源
2. 选择图表类型：柱状图、折线图、散点图、箱线图、直方图、热图
3. 映射数据列：x 轴、y 轴、分组列、颜色列
4. 点击"生成图表"将图表添加到画布

### 9.4 编辑图表

- 选中图表后属性面板显示图表专用编辑区
- 可修改：标题、轴标签、图例位置、配色方案、网格线
- 图表支持两种状态：
  - **数据绑定**：可重新加载数据，样式编辑受限于图表规格
  - **矢量化**：转为可编辑矢量组，可独立编辑每个元素，但数据绑定失效

### 9.5 图表转矢量

1. 选中图表元素
2. 右键 > 转为矢量组
3. 确认后图表被拆解为独立的 shape 和 text 元素
4. 每个柱子/线条/标签可独立编辑

## 10. 导出

### 10.1 导出 SVG

1. 文件 > 导出 > SVG
2. 导出选项：
   - 区域：当前视口 / 选中区域 / 完整画布
   - 背景色：白色 / 透明 / 自定义
   - 边距
3. SVG 保留矢量和文本信息，可在矢量编辑器中继续编辑

### 10.2 导出 PNG/JPG

1. 文件 > 导出 > PNG 或 JPG
2. 导出选项：
   - 倍率：1x / 2x / 3x
   - DPI：72 / 150 / 300
   - 背景色
   - 透明背景（仅 PNG）
3. 使用浏览器 Canvas 将 SVG 光栅化

### 10.3 导出 ZIP

- 文件 > 导出 > ZIP
- 打包 scene.json、data/、assets/ 为 ZIP 文件
- 导出的 ZIP 可被其他设备导入

## 12. 注意事项

1. **图层重叠**：移动或创建元素时若与同层其他元素重叠，操作会被阻止。可将元素移动到不同图层解决冲突。
2. **连接线**：连接线不受同层不可重叠约束，可跨层连接不同元素。
3. **浏览器兼容**：File System Access API（打开目录）仅在 Chrome/Edge 中支持，Firefox 使用 ZIP 导入方式。
4. **SVG 安全**：导入的 SVG 会自动清洗危险元素（script、事件处理器等）。
5. **离线使用**：PWA 模式下可离线打开已加载项目、编辑和导出。
